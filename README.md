# üìò **ChatGPT Python API Project**

## üöÄ Overview

This project is a **Python API chatbot** built using **FastAPI** and the **OpenAI API**.
It exposes simple HTTP endpoints that allow a client to send a message and receive a response generated by an OpenAI language model.

The API can be used for:

* Chatbots
* Assistants
* Simple natural language interfaces
* Educational or prototype projects

By default, free OpenAI accounts can only use a limited set of models (e.g., `gpt-4o-mini`).
Paid accounts will have access to more advanced models.

---

## üß† How the API Works

### **Endpoints**

#### `POST /chat`

This endpoint receives a JSON body:

```json
{
  "message": "Hello, chatbot!"
}
```

The server sends the message to the OpenAI API and returns the model‚Äôs response:

```json
{
  "reply": "Hello! How can I help you today?"
}
```

### **Flow**

1. The user sends a message to the API
2. The FastAPI backend validates the request using **Pydantic models**
3. The message is sent to an OpenAI model
4. The model generates a response
5. The API returns the response to the user
